# **App Name**: Giovanna's Digital Portfolio

## Core Features:

- Personalized Content Display: Showcase Giovanna Fernandes' information including profile picture, name, skills and a brief introduction, tailoring the content to her current role as a Computer Science student and Front-End Developer in training.
- Interactive Project Gallery: Enable the display of Giovanna's projects with details about each project, technologies used, and outcomes. Allow users to click and view project details.
- Dynamic Skills Section: Implement a dynamically updating section for listing Giovanna’s tech skills and personal interests, enabling easy adjustments and updates.
- About Me Section: Present Giovanna’s background focusing on her Computer Science journey, including her passion for technology and her front-end development goals. Use sections like 'The Beginning of a Passion for Technology', 'Focus on Front-End Development and UX', etc.
- Academic & Projects Timeline: A timeline showing the education, projects, and achievements of Giovanna during her Computer Science degree.

## Style Guidelines:

- Primary color: Vibrant pink (#E91E63) to create an energetic and inviting feel.
- Background color: Dark purple (#301934) for a modern and sophisticated dark theme.
- Accent color: A lighter, complementary shade of purple (#8E44AD) to highlight interactive elements.
- Headline font: 'Belleza' sans-serif for a stylish look
- Body font: 'Alegreya' serif for readability and elegance
- Use a set of elegant icons that reflect technology, development, and creativity in pink and purple tones.
- Recreate the original portfolio layout with identical element arrangement, optimized for showcasing skills and projects effectively.
- Subtle transition effects to navigate the single-page layout.
